أنت مصمم واجهات أمامية سينمائي (Creative Frontend Director).

المطلوب:
إضافة طبقة CSS أيقونية (Enhancement Layer) فوق الموقع الحالي
بدون تغيير الهيكل أو الـ HTML أو JavaScript.

الهدف:
رفع الإحساس البصري والحركي للموقع ليشبه:
افتتاحية فيلم ساموراي / لعبة AAA / علامة فاخرة يابانية.

================================
أفكار أيقونية يجب تنفيذها بـ CSS فقط
================================

1️⃣ إحساس السيف (Katana Feel)
- جميع الـ hover تكون سريعة وحادة (0.1s – 0.25s)
- استخدام clip-path لمحاكاة قطع بالسيف
- خطوط تتحرك كأنها أثر ضربة
- micro-shake خفيف عند الضغط (impact)

2️⃣ الحبر الياباني (Ink Effect)
- ظهور العناصر باستخدام mask-image كحبر يتمدد
- انتقالات غير منتظمة (organic)
- خلفيات كأنها فرشاة يابانية متحركة
- طبقة ink-overlay خفيفة تتحرك ببطء

3️⃣ الدخان والضباب
- fog layer باستخدام gradients + blur
- يتحرك ببطء مستقل عن المحتوى
- يتكثف عند الـ hover
- opacity يتغير حسب scroll

4️⃣ العمق السينمائي
- استخدام perspective + transform-style
- عناصر foreground تتحرك أسرع من الخلفية
- إحساس ثلاثي الأبعاد بدون 3D حقيقي
- parallax خفيف جداً

5️⃣ الضوء والصلب
- لمعان معدني يمر على الأزرار والعناوين
- انعكاس steel sweep عند hover
- glow ذهبي خافت وليس neon قوي
- استخدام mix-blend-mode للضوء

6️⃣ النصوص (Typography Motion)
- العناوين تظهر بضربة سيف واحدة
- حروف تتباعد ثم تستقر
- stroke أحمر خفيف خلف النص
- ظل متحرك كأنه ضوء نار

7️⃣ مؤشر الفأرة
- تغيير cursor حسب السياق:
  - زر → katana
  - قائمة → نقطة حمراء
- trail خفيف خلف المؤشر
- تفاعل مع hover (يتوهج)

8️⃣ Scroll Experience
- العناصر لا تظهر دفعة واحدة
- reveal متدرج (smoke → fade → slide)
- الحركة تعتمد على اتجاه التمرير
- إحساس وزن (heavy motion)

9️⃣ الجو العام
- film grain متحرك خفيف
- vignette داكن حول الشاشة
- لا يوجد لون أبيض صريح
- الأسود غير نقي (cinematic black)

================================
قيود مهمة
================================
- CSS فقط (لا JS جديد)
- لا تغيير في الـ HTML
- تحسينات إضافية فقط
- أداء عالي
- دعم prefers-reduced-motion

================================
المخرجات
================================
- أفكار مطبقة في CSS
- Keyframes مخصصة
- متغيرات CSS منظمة
- تعليقات توضح الفكرة البصرية
- أسلوب نظيف وقابل للتوسع

النتيجة النهائية:
موقع يشعر المستخدم أنه "تم قطعه وصقله بالسيف"
وليس مجرد موقع عادي.
